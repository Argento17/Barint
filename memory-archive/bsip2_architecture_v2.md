BSIP2 v2 architecture defined 2026-05-18. Location: `01_framework\bsip2_framework\architecture_v2\`

**Core design shift:** From "NOVA + modifiers (single weighted-dimension pool)" to "four interpretive layers + negotiated orchestration."

**The four layers:**
1. **Structural Integrity** — "Does this still behave like food?" NOVA lives HERE, inside L1, not above the system. Signals: NOVA, reconstruction intensity, ingredient topology, additive architecture, matrix coherence, fermentation credit.
2. **Nutritional Contribution** — "Does this materially nourish?" Positive-oriented. Fortification skepticism built in. Signals: protein delivery+quality, fiber, macro coherence, protein realism.
3. **Metabolic Stability** — "How physiologically stable is this?" Glycemic plausibility, satiety plausibility, liquid calorie effect, calorie density in context.
4. **Consumption Engineering** — "Was this designed for overconsumption?" Inverted layer (higher = more concern). Hyper-palatability, sweetener systems, flavor amplification, reward stacking.

**Orchestration (NOT averaging):**
- L1 sets structural CEILING (low integrity = cap on max score)
- L2 sets nutritional FLOOR (high contribution = minimum score protection)
- L3 adjusts ±15 pts within ceiling/floor range
- L4 applies multiplicative downward pressure (0–45% reduction)
- Regulatory signals (red labels, trans fat veto) applied outside layers as hard policy

**Key collinearity fix:** Current processing_quality + whole_food_integrity + additive_quality = 29% weight but all driven by NOVA. In v2, all absorbed into L1 as one voice. NOVA's effective dominance reduced.

**Key protein collinearity fix:** nutrient_density (65% protein) + protein_quality + satiety_support = 31% weight, all protein-heavy. In v2, L2 evaluates protein once with source quality built in; L3 evaluates satiety composite.

**Contradiction types (preserved, visible in output):**
- C-1: Strong nutrition + poor structure (protein isolate drinks)
- C-2: Strong structure + limited nutrition (whole-food oils)
- C-3: Metabolic stability + engineering presence
- C-4: Fortification + structural reconstruction (breakfast cereals)

**Transition phases:** 7 phases. Currently Phase 0 (architecture docs). No code changes yet. Shadow mode in Phases 2-3. Full cutover in Phase 5.

**Bari identity:** "Structural food intelligence" — NOT nutrient thresholds, NOT pure NOVA, NOT anti-processing, NOT low-calorie optimization, NOT diet recommendations.

**Why:** How to apply: any future scoring proposal must answer "does this make structural food intelligence more accurate, honest, or explainable?" — not "does this produce scores that feel right?"
