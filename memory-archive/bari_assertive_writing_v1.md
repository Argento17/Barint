Built 2026-05-27. Master document at `C:\Bari\01_framework\editorial\assertive_writing_v1.md`.

**Why:** The governance layer solved overclaiming. A secondary problem emerged: writing became over-hedged, over-qualified, and editorially weak. This document defines the floor — what Bari must say, and how to say it with force.

**How to apply:** governance_v1.md defines the ceiling (what Bari cannot say). This document defines the floor (how assertive the writing should be). Both apply to all consumer-facing output.

## Core Rule

Hedging = epistemic insecurity (bad). Restraint = scope discipline (good). Bari has zero hedging and complete restraint.

## Ten Components

1. **Assertive but Restrained** — spectrum table: defensive/over-hedged vs. target vs. aggressive. The rule: when evidence is direct, state it directly.
2. **Confidence Map** — Layer 1 (observation) = full confidence. Layer 2 (interpretation) = bounded confidence. Beyond Layer 3 = no output. Strong confidence domains: ingredient order, presence, score, transparency. Uncertainty required: manufacturer intent, process, health outcomes.
3. **Finding-First Writing** — 3 templates: Template A (finding + evidence), B (finding + evidence + limit), C (partial evidence). Anti-throat-clearing rule.
4. **Phrase Library (130+)** — 8 categories: comparison/contrast, ingredient observation, verification/evidence, transparency/availability, score/analysis, pattern/distribution, fermentation-specific, grain/fiber-specific. Plus: over-hedged forbidden list, too-aggressive forbidden list.
5. **Investigative Tension** — 6 safe tension sources: same name/different composition, score divergence, premium vs simple, name vs primary ingredient, transparency gap, label ambiguity. Article/comparison/headline/insight-block examples for each.
6. **Assertive Score Explanation** — specificity rule, good vs weak table, causal chain rule (observable signal → score effect), anti-genericity test, short template formats.
7. **Confident Transparency** — transparency is editorial posture, not apology. Phrasing system, UX wording, matter-of-fact register vs apologetic register.
8. **No Apology Rule** — 7 forbidden apologetic phrases removed. One footer disclaimer; zero body-copy disclaimers. No-apology test: if disclaimer sentences > finding sentences → rewrite.
9. **High-Signal Writing** — 5 low-signal patterns to eliminate. Rewrite examples. Paragraph test. Density target: every sentence names a product/ingredient/score/measurement.
10. **Journalistic Identity** — archetype: investigative market correspondent, not food blogger or nutritionist. 8 identity principles. 6 "never" voices. The north star sentence: "ברי בדקה את המדף בזהירות. הנה מה שמצאה."

## Key Rules to Remember

- State observations directly. Mark inferences explicitly.
- Finding first. Evidence second. Limitation last (when needed).
- One footer disclaimer. Zero body-copy disclaimers.
- "אולי ייתכן שכנראה" is forbidden. "לא ניתן לאמת X" is correct.
- Score explanations must name a specific ingredient, position, or value.
