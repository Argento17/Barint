## Reporting Lines

```
Tom ──── ChatGPT (strategic direction)
Tom ──── Claude CE (content authority + framework)
Tom ──── OpenAI Codex (audit escalation)
ChatGPT ──── Claude CE (experience spec → pipeline brief)
Claude CE ──── Cursor IDE (JSON + handoff spec)
OpenAI Codex ─ ─ ─ all roles (independent audit, dashed line)
```

**Dashed audit line convention:**  
OpenAI Codex has independent read access to all roles' outputs. It does not report through the hierarchy for audit findings — it reports directly to Tom when critical.
